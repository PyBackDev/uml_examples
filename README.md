## Installation

1. Clone repository
```console
git clone git@github.com:PyBackDev/uml_examples.git
```

2. Install hatch package manager
```console
pip install hatch
```

3. Set up environment
```console
hatch env create style
hatch shell style
```

4. Run code check
```console
hatch run style:lint
```
